Welcome to **Common Kubernetes Pod Failure Scenarios**! This repository contains YAML files and examples to simulate and troubleshoot three common Kubernetes pod failures: image pull errors, resource limits exceeded, and application errors. Perfect for learning Kubernetes troubleshooting hands-on!

## Whatâ€™s Inside?

This repo includes three YAML files that intentionally trigger pod failures, along with instructions to reproduce and resolve them using `kubectl`. The scenarios are:
1. **Image Pull Error**: Misconfigured image name.
2. **Resource Limits Exceeded**: Tight CPU/memory limits causing crashes.
3. **Application Error**: App exits with an error code.
